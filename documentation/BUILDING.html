Notes on building the library

In order to build the library, you need a gcc compiler (a cross compiler is reccomended) and the AmigaOS4 SDK. Everything is already included if you download the cross compiler suite from https://github.com/sba1/adtools.
Follow the instructions on adtools site and you are ready to go

There is a makefiles for GCC called(GNUmakefile.os4). The file is named in this way because the old clib4 version that had both .os3 and .os4 versions and adtools is searching for it.
So to compile the lib just do a simply: 

make -f GNUmakefile.os4

and the ibrary will be built
